# Biblioteca 🌸

Sistema simples de controle de estoque para uma biblioteca, feito em Java como projeto de estudo.

## Sobre o projeto

O sistema permite cadastrar livros, controlar a quantidade disponível e o valor de aluguel de cada item.

## Estrutura

- `Produto` — representa um livro do acervo (nome, quantidade e valor de aluguel).
- `Estoque` — gerencia a lista de produtos e concentra as operações do sistema.
- `Main` — interface de console para o usuário interagir com o estoque.

## Funcionalidades

- Adicionar produto
- Listar produtos
- Buscar produto por nome
- Atualizar quantidade em estoque
- Remover produto

## O que estou aprendendo

Este projeto foi feito como prática de Java para reforçar conceitos básicos de orientação a objetos, entre eles:

- Conceitos de orientação a objetos: classes, encapsulamento (atributos privados com getters e setters), construtores (com e sem parâmetros), ArrayList (para armazenar coleções de objetos) e manipulação de coleções
- Criação de classes e separação de responsabilidades
- Laços de repetição (for-each e while) para percorrer listas e controlar o menu
- Leitura de dados do usuário com Scanner

## Como rodar

1. Clone o repositório
2. Abra o projeto no IntelliJ (ou outra IDE)
3. Execute a classe `Main`
